1. What is the Document Object Model (DOM)?

DOM or Document Object Model is a way for JS to make the website more dynamic by being interactive. It illutrates the structure of the webpage as a tree of objects, where every part is a object of that page. It makes obtaining the parts easier.

2. How do you select an element from the DOM?

Select An Element From The DOM:
We can select an element by mostly 4 methods:
1.Select By Id:
Use document.getElementById() to select a single element with a specific id.
2.Select By Class-Name:
Use document.getElementsByClassName() to select all elements with a specific class.
3.Select By Tag-Name:
Use document.getElementsByTagName() to select all elements with a specific tag name.
4.Select By CSS-selectors:
Use document.querySelector() to select the first element that matches a specified CSS selector, or document. querySelectorAll() to select all matching elements.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation: Event delagation is the way of handling events which is used for attaching a listener to parent instead of their children . It hears all event and gives appropriate results. An example: document.getElementById('parent-list').addEventListener('click', function(event) { if (event.target.tagName === 'LI') { alert('You clicked on ' + event.target.textContent); } }); Why is it useful? Less code: You only need one event listener on the parent element, instead of adding listeners to each button, link, or item. Better performance: Attaching many listeners can slow down the webpage. Event delegation uses just one listener for multiple elements, so the page runs faster. Handles new elements: If new elements are added dynamically (after the page loads), the parent can still detect events on them, without adding extra listeners.
